# Copymantle

I played [Semantle](https://semantle.com/) all the time, unfortunately
it was acquired by some company that barely maintained it, and the
server got really buggy. I think I tried to report some bugs and never
really heard back or saw them fixed, so on the n'th time that the
official instance bugged out while I was trying to play with friends,
I was like "this cannot be more than an hour or two of coding" and
just threw together something that more or less replicated the same
experience.

There isn't an official instance of this that I host for the public to
use, but you could run your own if you would like.
